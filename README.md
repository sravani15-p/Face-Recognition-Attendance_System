# Face Recognition Attendance System

## Problem Statement
Manual attendance is time-consuming and allows proxy attendance, which affects accuracy.
## Solution
This system uses face recognition to automatically identify students and mark attendance without manual intervention.
## Working
1. Camera captures student face.
2. Face is detected using OpenCV.
3. Face is compared with stored database.
4. If matched, attendance is marked.
5. Data is saved securely.
## Technologies Used
- Python
- OpenCV
- Camera Module
## Applications
- Schools
- Colleges
- Offices
